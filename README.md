## OnlyFood 
Food Order App For Small Restaurant
## Features
- Authentication OTP
- Food Searching
- Filter food
- Order food
- Add to cart
- Like food
- Category 
## Tech 

**Client:** Android Studio (Java)

**Server:** Nodejs

**Database:** MongoDb
